# YDT Vocabulary Trainer

A vocabulary learning project that combines word-data preparation scripts with a browser-based study application designed for repeated practice.

## Overview

This workspace goes beyond a simple word app. It includes both the learning interface and the data processing scripts used to clean, extract, and organize vocabulary resources. That makes it a stronger portfolio project because it shows both product thinking and supporting data workflow.

## Highlights

- level-based vocabulary study flow
- persistent learning progress with local storage
- unknown-word review mode
- PWA-ready frontend structure
- supporting Python scripts for vocabulary extraction and cleanup
- reusable JSON-based dataset generation

## Tech Stack

- JavaScript
- HTML
- CSS
- JSON
- localStorage
- Python

Open the files inside `vocab-app` in a browser or serve them with a local static server.

## What This Project Demonstrates

- building a study product around a specific educational need
- working with raw learning data and transforming it into usable app content
- structuring content into levels and sessions
- maintaining a simple but effective offline-friendly frontend
