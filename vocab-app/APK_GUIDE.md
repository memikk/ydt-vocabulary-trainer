# How to Create Your YDT Sözlük APK

Your vocabulary app is built as a **Progressive Web App (PWA)**. This means it runs in any browser but can also be converted into a real Android App (APK).

### Method: Use "Website 2 APK" or Online Builder
1.  I have created a file named **`YDT_Sozluk_App.zip`** on your Desktop/Folder.
2.  This file contains everything: the icon, the dictionary data, and the code.
3.  <a href="https://websitetoapk.com/" target="_blank">Download Website 2 APK Builder</a> (Windows) OR search "Web to APK converter online".
4.  Load the **`vocab-app` folder** (or the bits inside the zip).
5.  Click **Build**.

## Option 2: Run in Browser (No APK needed)
You don't *need* an APK to use it on your phone!

1.  Upload the `vocab-app` folder to any free hosting (Netlify, GitHub Pages, etc.).
2.  Open the URL on your phone (Chrome/Safari).
3.  Tap **Menu (3 dots) -> Add to Home Screen**.
4.  It will install like an app!

## Option 3: Local Testing
Just open `index.html` in Chrome/Edge on your PC. It works offline because we embedded the data!
