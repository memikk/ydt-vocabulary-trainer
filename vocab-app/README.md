# Vocabulary App

A lightweight PWA-style vocabulary trainer focused on repetition, level progression, and persistent study history.

## Features

- level-based study cards
- review mode for unknown words
- local progress tracking
- dashboard view for completion status
- web app manifest for installable usage

## Tech Stack

- JavaScript
- HTML
- CSS
- JSON
- localStorage

## Files

```text
vocab-app/
  index.html
  app.js
  style.css
  data.js
  manifest.json
```

## Running Locally

Open `index.html` in a browser or use a simple static server.

## Suggested Repository Name

`vocabulary-app`
